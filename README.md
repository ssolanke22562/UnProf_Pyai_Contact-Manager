# 📖 JSON Contact Manager CLI

A lightweight Command Line Interface (CLI) application built in Python for managing personal or professional contacts. This tool runs entirely in the terminal and uses a local JSON file for persistent, human-readable data storage, meaning no external database setup is required.

### ✨ Features
* **Add Contacts:** Store names, phone numbers, emails, and custom notes.
* **Persistent Storage:** Data is automatically saved and loaded from a cleanly formatted `contacts.json` file.
* **Smart Search:** Search across all fields at once (find a contact by their name, a snippet of their email, or a keyword in their notes).
* **Dependency-Free:** Built using only Python's standard library (`json`, `os`).
